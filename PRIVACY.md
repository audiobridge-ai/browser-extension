# Privacy Policy – AudioBridge

**Last updated**: 2026-01-02

AudioBridge respects your privacy.

**Data Collection**

AudioBridge does not collect, store, or transmit user data to any developer-controlled servers.

**Data Processing**

The extension processes the following data solely to provide its functionality:

- Audio recordings from the user’s PLAUD account

- Transcription text generated using the user’s selected API

- User-provided API configuration (API key and endpoint)

All processing is initiated by the user and performed locally or directly between the user’s browser and the selected API provider.

**Data Storage**

Any configuration data or temporary files are stored locally on the user’s device and can be removed by uninstalling the extension.

**Third-Party Services**

AudioBridge communicates only with:

- PLAUD web services

- The OpenAI or OpenAI-compatible API endpoint specified by the user

AudioBridge does not use analytics, tracking, or advertising services.

**Contact**

If you have questions about this privacy policy, please contact us via:
https://github.com/audiobridge-ai/browser-extension/issues
